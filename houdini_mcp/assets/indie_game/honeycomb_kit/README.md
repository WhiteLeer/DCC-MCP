# Honeycomb Kit (Indie Game)

This folder contains Houdini assets created for hex-based indie game world building.

## Files
- `hex_bevel_block.hda`: Base beveled hex cell unit.
- `hex_stair_set.hda`: Stair transition module aligned to base cell scale.
- `hex_cliff_half_cell.hda`: Half-cell cliff module with jagged cliff-wall controls.
- `honeycomb_hexmap_build.hip`: Working scene for iterative authoring and testing.

## Notes
- These assets are intended for modular level construction and export to engine pipelines.
- Keep this folder as the canonical source for the current generated kit.
