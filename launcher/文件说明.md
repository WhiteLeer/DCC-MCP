## 📄 核心文件

### ⭐ 启动 Claude Code.bat
**类型：** 批处理脚本
**用途：** 主启动入口
**内容：**
```batch
@echo off
cd /d "%~dp0"
python "启动Claude Code.py"
```

**使用：**
```bash
双击启动，显示启动器 GUI
```

---

### ⭐ 启动Claude Code.py
**类型：** Python 脚本
**用途：** 启动器核心逻辑
**功能：**
- 锁文件检测
- 窗口标题修改
- 实例编号管理
- 插件启用/禁用
- GUI 界面

**关键函数：**
```python
is_houdini_mcp_running()        # 锁文件检测
get_next_normal_instance_number()  # 获取编号
set_window_title()              # 修改标题
enable_houdini_plugin()         # 切换插件
launch_claude()                 # 启动 Claude Code
```

## 🔧 安装和配置

### 安装依赖.bat
**用途：** 安装 Python 依赖
**依赖：**
- `pywin32>=305`：修改窗口标题
- `psutil>=5.9.0`：进程检测

**命令：**
```bash
python -m pip install -r requirements.txt
```

### requirements.txt
**内容：**
```
pywin32>=305
psutil>=5.9.0
```

### mcp.json
**类型：** MCP 配置文件
**用途：** Claude Code 插件配置
**位置：** 需复制到 `~/.claude/plugins/houdini-mcp/`

## 🗂️ 用户数据文件

### 锁文件
**路径：** `~/.claude/plugins/houdini-mcp/.running.lock`
**用途：** 标记 Houdini MCP 运行状态
**内容：**
```
pid=12345
started=1234567890.123
```

### 实例计数器
**路径：** `~/.claude/.normal_instance_counter`
**用途：** 普通模式实例编号
**内容：**
```
3
```
表示下次启动将是"普通-3"

## 📋 使用流程

### 首次使用
```
1. 双击：安装依赖.bat
   ↓
2. 双击：测试锁文件检测.bat （可选）
   ↓
3. 双击：启动 Claude Code.bat
   ↓
4. 选择模式，开始使用
```

### 日常使用
```
双击：启动 Claude Code.bat
  ↓
选择模式
  ├─ 普通模式 → 日常开发
  └─ MCP 模式 → 3D 工作
```

### 调试问题
```
1. 双击：测试锁文件检测.bat
   ↓
2. 检查锁文件状态
   ↓
3. 查看：功能说明.txt → 故障排除
```
