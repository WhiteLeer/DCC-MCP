## 💡 调试提示

### 如果工具没有出现
1. 检查 `claude_desktop_config.json` 格式
2. 重启Claude Code
3. 查看错误日志（Help → Show Logs）

### 如果调用失败
1. 使用 `get_scene_state` 查看当前状态
2. 检查节点路径是否正确
3. 查看详细错误信息

### 验证配置
